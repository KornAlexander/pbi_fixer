You have 100 reports in a workspace. All need the same fix — standardize page size, hide visual filters, apply BPA rules.

Are you really going to open each one individually?

The PBI Fixer supports multi-report AND multi-model editing:

- Comma-separated input: `"Sales Report, Finance Report, HR Dashboard"`
- Leave blank → loads ALL reports and models in the workspace
- Unified tree view — all reports/models visible at once
- Apply fixers across all loaded items in one pass

Run "Fix All" on 100 reports simultaneously. Apply BPA fixes to a few semantic models at once.

This turns a full afternoon of repetitive clicking into one operation.

The tree view merges everything — navigate between reports and models without reloading. Click a visual in Report A, then switch to Report B. It's all there.

Batch operations at workspace scale. The way it should be.
