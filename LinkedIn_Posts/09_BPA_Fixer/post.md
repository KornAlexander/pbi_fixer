# LinkedIn Post — BPA: Scripts Are Great, But...

**Image:** model-bpa.png (+ report-bpa.png as second image if LinkedIn carousel)

---

BPA rules are great. 19 of them even have auto-fix scripts.

But be honest — do you know how to apply them?

Most people I talk to run the Best Practice Analyzer, see the violations, and then... fix them manually. One by one.

The scripts exist. They're powerful. But applying them requires:
- Knowing which script fixes which rule
- Running them in the right order
- Passing the correct parameters
- Handling edge cases (like measures already having format strings)

The PBI Fixer wraps all 19 fix scripts into a single workflow:

1. Runs the Model BPA → gets all violations
2. Maps each violation to its fixer script automatically
3. Applies all fixes in bulk
4. Reports what was fixed and what's informational-only

Plus: 4 Report BPA rules with auto-fix (page size, hidden filters, etc.)

"Fix All" — one button. That's it.

15 of these 19 fixes have NO standard auto-fix in the default BPA rules. They only exist here.

Try it: 🎯 https://actionablereporting.com/pbi-fixer
